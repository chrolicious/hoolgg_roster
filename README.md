## Quick Start

1. Download and run the installer
2. Launch **Hool.gg //Roster** from your desktop
3. Add your Blizzard API credentials in **Settings** (optional but recommended)
4. Configure your characters and start tracking!

## Getting API Credentials

1. Visit [develop.battle.net](https://develop.battle.net/)
2. Create a new OAuth client (use `http://localhost` as redirect URL)
3. Copy your Client ID and Client Secret
4. Enter them in the Settings tab
